## 📈 Proyectos destacados

### 🗞️ [Análisis_seminario](https://github.com/jahadd/Analisis_seminario)
Repositorio del seminario de título (Magíster en Psicología Social, UAI).

Incluye dos componentes:
- **`Prosocial_marinos/`** — Análisis experimental del esfuerzo prosocial en grupos socialmente diferenciados (marinos APOLINAV, vulnerables, control). ANOVAs de medidas repetidas, DASS-21, scripts R para visualización.
- **`analisis_noticias/`** — *Media scoping review* de prensa chilena (2010–2025): scraping, filtro de corpus por capas de pertinencia, análisis de redes temáticas y series temporales.

> **Stack:** R · JASP · PostgreSQL · ggplot2
